{Node} domDrop
the DOM element users drag files into.
{String} descType
a string describing what type of files can be uploaded by this uploader. Ex: "png, jpg, or gif".
{String} objectName
the name of the directory the objects go in, ie: "images"
{RegExp} fileRestriction
a regular expression filetypes must pass to be allowed to be uploaded by this service.
{Node} domBubble

{Node} domBtn
