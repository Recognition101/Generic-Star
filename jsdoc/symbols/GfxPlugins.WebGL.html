loadImage(img, callback, tile)
This function loads an image into OpenGL Memory. Note that if you are using the scaffolding, this is all taken care of by the Sprite Handler, and you should probably be using getImage to get sprites for drawing.
Parameters:
{String} img
the filename of the image to load in
{Function} callback
{(GfxPlugins.WebGL.Texture):()} this function is called upon completion of loading
{Boolean} tile Optional
true if you plan on tiling this image. If true, the image must have width and height values that are powers of two.
