showButtonDialog(title, text, btnArr, callback)

Parameters:
{String} title
the title that the dialog will have
{String} text
the text that is displayed under the title
{Array} btnArr
an array of strings with each button name
{Function} callback

showImageDialog(defaults, callback)
Brings up an image selection dialog.
Parameters:
{Object} defaults
the defaults to set this dialog to (url, worldAlign, animated, frameCol, frameRow, startFrame, endFrame).
{Function} callback
a function that will be called back upon clicking the "save" button. Note calling showImageDialog does not guarantee that callback will be called. callback takes 1 argument, which is an object describing the settings set by the dialog.
